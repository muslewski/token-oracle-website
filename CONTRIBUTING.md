# Contributing to token-oracle-website

This repository is the **marketing one-pager** for [`token-oracle`](https://github.com/muslewski/token-oracle).

## Where to file what

| Topic | Repo |
|---|---|
| Tool bugs, CLI features, install failures | **[token-oracle](https://github.com/muslewski/token-oracle/issues/new/choose)** |
| Questions & ideas about the tool | **[token-oracle Discussions](https://github.com/muslewski/token-oracle/discussions)** |
| Website copy / layout / performance | Issues on **this** repo |

## Dev (website)

```bash
npm install
npm run dev      # Vite
npm run build
```

Please keep product install commands accurate (`npm install -g token-oracle` / `npx …`).
