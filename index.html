04 · quickstart

Run the oracle

npm, npx, uv, or pip — same offline forecast. Click to copy.

setup
$ npm install -g token-oracle ⎘
#1 Node shim — bins: token-oracle · oracle (runs uvx/pipx under the hood)
$ token-oracle init ⎘
#2 Guided setup (or --preset max20)
$ token-oracle dash ⎘
#3 Full-screen past / present / future
$ token-oracle forecast ⎘
#4 Time left before your cap
$ npx token-oracle dash ⎘
#1 Instant — no global install
$ npx token-oracle forecast ⎘
#2 One-shot forecast
$ npx token-oracle init ⎘
#3 Configure profiles
$ uvx token-oracle dash ⎘
#1 Ephemeral Python runner (recommended pure path)
$ uv tool install token-oracle ⎘
#2 Install onto PATH via uv
$ token-oracle forecast ⎘
#3 Same CLI after install
$ pipx install token-oracle ⎘
#1 Isolated Python install
$ pip install token-oracle ⎘
#2 Into the current environment
$ token-oracle doctor ⎘
#3 Check config + data sources
✓ Real tool is Python on PyPI — npm/npx is a thin launcher.
